## Instrukcije za pokretanje fajlova

Python fajlovi se pokreću direktno iz komandne linije. Pristupite direktorijumu u kom se nalaze fajlovi i izvršite sledeću komandu: `python P-*.**.py` 

**Napomena**: `*` karaktere zamenite odgovarajućim brojevima koji se nalaze u nazivu fajla koji testirate.
